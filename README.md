# Golden Gift Palace

Welcome to the Golden Gift Palace website! This is a beautifully designed showcase website for our diverse range of products including cosmetics, toys, gifts, and religious items.

## About

Golden Gift Palace is your one-stop shop for premium products. We offer:

- **Cosmetic Products:** Lipsticks, foundations, mascaras, hair care, skincare, jewelry, perfumes, and more
- **Toys and Gifts for Kids:** Age-appropriate toys, educational games, and gifts for children aged 0-15
- **Religious Items:** God idols, photo arts, and decorative frames

## Features

- **Elegant Navigation:** Fixed navigation bar with smooth scrolling to different sections
- **Beautiful Design:** Warm, luxurious color scheme with gold accents perfect for a gift store
- **Product Categories:** Organized display of products across multiple categories with attractive cards
- **About Section:** Information about our store with decorative elements
- **Contact Information:** Complete contact details including phone, email, address, and social media
- **Responsive Design:** Optimized for all devices with mobile-first approach
- **Smooth Animations:** Hover effects, transitions, and interactive elements throughout

## Design Highlights

- **Color Scheme:** Warm gold and brown tones evoking luxury and elegance
- **Typography:** Custom Google Fonts (Poppins and Playfair Display) for modern readability
- **Visual Effects:** Gradients, shadows, and subtle animations
- **Card Design:** Elevated product cards with hover animations
- **Layout:** Clean, spacious design with proper visual hierarchy

## Getting Started

To run this website locally:

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open your browser to `http://localhost:5173`

## Technologies Used

- React
- Vite
- CSS3 with custom properties
- Google Fonts
- Modern CSS features (gradients, backdrop-filter, etc.)
